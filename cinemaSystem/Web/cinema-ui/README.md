# Cinema UI

Frontend for the Cinema System, built with React + TypeScript + Vite.

## Scripts
- `npm run dev`: run local dev server
- `npm run build`: type-check and build
- `npm run lint`: run eslint
- `npm run preview`: preview production build

## Environment
- `VITE_API_URL`: backend base URL (default in code: `https://localhost:7227`)
